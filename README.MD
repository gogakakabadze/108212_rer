## Overview

This MLOps platform automates YOLO model training on AWS SageMaker with:

- **Automated triggers** - S3 upload or GitHub push starts training
- **Experiment tracking** - MLflow logs all metrics and artifacts
- **Model versioning** - Models registered in SageMaker Model Registry
- **Infrastructure as Code** - AWS CDK manages all resources

| Step | Action | Component | Description |
|------|--------|-----------|-------------|
| 1 | Dataset Upload | S3 | Upload dataset to `s3://bucket/split_dataset/` |
| 2 | Event Detected | EventBridge | Listens for S3 object creation |
| 3 | Lambda Invoked | Lambda | Validates dataset structure |
| 4 | Pipeline Started | SageMaker | Creates training job on GPU instance |
| 5 | Data Downloaded | Training Job | Copies dataset from S3 to instance |
| 6 | YOLO Training | Training Job | Runs `train2.py` with YOLOv11 |
| 7 | Metrics Logged | MLflow | Logs loss, mAP, epochs, etc. |
| 8 | Model Saved | S3 | Uploads `model.pt` to artifacts bucket |
| 9 | Notification | SNS | Sends training completion alert |

### Naming Convention

All resources follow this pattern:

```
{project_name}-{resource-type}-{account_id}
```

Example with `project_name = athmech-mlops`:

| Resource | Name |
|----------|------|
| Training Bucket | `athmech-mlops-training-data-637423443220` |
| Artifacts Bucket | `athmech-mlops-model-artifacts-637423443220` |
| MLflow Bucket | `athmech-mlops-mlflow-artifacts-637423443220` |
| SageMaker Role | `athmech-mlops-sagemaker-execution-role` |
| Lambda Role | `athmech-mlops-lambda-execution-role` |
| Pipeline | `athmech-mlops-yolo-training-pipeline` |
| Lambda | `athmech-mlops-training-trigger` |

## Quick Reference

### Deploy Infrastructure

```powershell
cd athmech-mlops-infra
cdk bootstrap
cdk deploy --all
```

### Trigger Training Manually

```bash

# Go to Actions â†’ Deploy and Train â†’ Run workflow

# Via AWS CLI

  --pipeline-name athmech-mlops-yolo-training-pipeline
```

### Download Trained Model

```bash
# Find latest execution
aws sagemaker list-pipeline-executions \
  --pipeline-name athmech-mlops-yolo-training-pipeline

# Download model artifacts
aws s3 cp s3://athmech-mlops-model-artifacts-{account}/pipelines/{execution-id}/output/model.tar.gz .
tar -xzf model.tar.gz
```

## Summary

| Component | Technology | Purpose |
|-----------|------------|---------|
| **Infrastructure** | AWS CDK | Deploy all AWS resources |
| **Training** | SageMaker Pipeline | Run YOLO training on GPU |
| **Tracking** | MLflow | Log metrics & experiments |
| **Triggers** | EventBridge + Lambda | Auto-train on S3 upload |
| **CI/CD** | GitHub Actions | Auto-train on code push |
| **Storage** | S3 | Datasets, models, artifacts |
| **Registry** | SageMaker Model Registry | Version trained models |
